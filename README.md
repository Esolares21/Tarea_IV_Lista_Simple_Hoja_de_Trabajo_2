feat: implementación de estructura de datos dinámica (Lista Enlazada Simple) 🚀

Descripción:
Este commit añade el desarrollo completo de una lista enlazada utilizando memoria dinámica en C++ para el curso de Programación 2. 💻

Funcionalidades principales:
* 🛠️ Definición de estructura 'Nodo' con punteros.
* 🧠 Uso de 'new' para la gestión de memoria dinámica.
* 🔄 Ciclo de ingreso indefinido controlado por el usuario.
* 📍 Recorrido detallado y visual de la lista (formato dato -> NULL).
* 📊 Algoritmos para cálculo de suma, conteo de nodos y promedio.

El código sirve como base para entender el manejo de punteros y la eficiencia en el almacenamiento de datos no contiguos. ✨
