# Tarea IV: Hoja de Trabajo 2

## Descripción del Proyecto y Funcionalidades

Este código representa la implementación de una Lista Enlazada Simple utilizando memoria dinámica. El objetivo principal es demostrar el control de punteros y la gestión de nodos en el "Heap". 💻

¿Qué hace el código?

Gestión Dinámica: Utiliza el operador new para crear espacios de memoria por cada nodo de forma indefinida. 🧠

Interacción con el Usuario: Implementa un ciclo while que permite al usuario decidir en tiempo real cuántos datos ingresar. 🔄

Procesamiento de Datos: Al finalizar la carga, el programa realiza un recorrido completo para mostrar los valores y calcular automáticamente:

La cantidad total de nodos creados. ✅

La suma acumulada de todos los valores. ➕

El promedio aritmético de los datos. 📊

Salida Visual: Genera una representación gráfica de la estructura en consola (ej. 10 -> 7 -> 25 -> NULL) para verificar la correcta conectividad de los punteros. 📍

¿Cómo se utiliza?

Al ejecutar, el sistema te pedirá ingresar un número entero.

Después de cada ingreso, podrás elegir entre continuar (s) o terminar el proceso (n).

Al finalizar, el programa imprimirá el reporte detallado y la estructura final de la lista. ✨
